## Quick start

1. Install deps: `npm install`
2. Run locally: `npm run dev`
3. Build: `npm run build`

## Contact form

The contact form in this template is wired to Formspree. Replace the `FORM_ACTION` in `components/Contact.jsx` with your Formspree endpoint.

## Deploy
Push to GitHub and import the repo on Vercel. Add `portfolio.trendify.co.ke` as a domain in Vercel and create a CNAME at your DNS provider pointing `portfolio` to the Vercel target.
